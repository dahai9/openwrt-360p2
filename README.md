# openwrt-360p2
This is a firmware for 360p2,it support xray.

这是一个支持xray的360p2固件
## use
Just download the '36.tar.gz' and unzip it choose you need file flash by u-boots.

下载‘36.tar.gz’--》解压--》选择需要的文件刷入即可
## attention 
由于360p2路由器只有16m闪存，只添加了默认的基础功能和ssrplus中的xray.
